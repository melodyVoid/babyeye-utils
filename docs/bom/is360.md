### 1.1 is360 判断浏览器是否为 360 浏览器

#### 描述 [Description]

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;用于判断浏览器是否为 360 浏览器

#### 用法 [Syntax]

> ```js
> is360()
> ```

#### 参数 [Parameters]

&nbsp;&nbsp; &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;**无**

#### 返回值 [Return value]

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;布尔值，是360浏览器为 `true`，否则为 `false`。

#### 例子 [Examples]

```js
// 用 Chrome 测试。
bUtils.is360(); // false
```
